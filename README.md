## 📂 Project Directory

### 1. 🏥 Disease Prediction & Intelligent Doctor Matcher
* **Files:** `Disease_Prediction_Model.ipynb`, `Copy_of_Disease_Prediction_Model...`,'https://huggingface.co/spaces/oce1230/Disease-Prediction-System'
* **Core Stack:** Python, Scikit-Learn, Pandas, NumPy
* **Description:** A dual-purpose healthcare application. First, it implements a classification pipeline to predict patient health risks based on clinical symptoms. Second, it features an automated recommendation engine designed to pair patients with localized medical specialists. 
* **Data Integrity Note:** To protect medical provider privacy and adhere to data consent best practices, the doctor database currently utilizes structurally sound mock/synthetic coordinates and contact details rather than unverified live listings.

### 2. 🏛️ UIDAI Demographic Data Engineering (National Hackathon)
* **Files:** `UIDAI_Hackathon.ipynb`
* **Core Stack:** Python, Matplotlib, Seaborn, Advanced Exploratory Data Analysis (EDA)
* **Description:** An in-depth behavioral and regional analysis of large-scale demographic data initiated during a national hackathon. This project uncovers high-impact macroeconomic anomalies across administrative datasets, with ongoing expansion to scale up processing for massive data volumes.
* **Key Algorithmic Insights Found:**
  * **"March Madness" Temporal Spike:** Isolated a massive 300% surge in activity during the month of March, driven by contrasting demographic pressures between students and adults.
  * **The "Two Indias" Regional Pattern:** Uncovered extreme operational divergence based on local state policies:
    * *The Student Hub (e.g., Haveri, Karnataka):* Driven by strict state mandates requiring mandatory Aadhaar seeding with Student IDs for the State Scholarship Portal (SSP).
    * *The Agricultural Hub (e.g., Buldhana, Maharashtra):* Driven by critical rural timelines requiring farmers to complete mandatory e-KYC updates to receive their PM-Kisan installments.

### 3. 📉 Customer Churn Prediction Engine (IBM SPSS)
* **Files:** `SPSS PROJECT.zip`
* **Core Stack:** IBM SPSS Modeler (Node-Based Visual Workflows)
* **Description:** An enterprise-focused predictive analytics project targeting corporate customer retention. By building and evaluating node-based stream architectures, this model maps out non-linear customer behaviors, isolates critical churn risk factors, and provides businesses with automated retention flags before a customer defects.
